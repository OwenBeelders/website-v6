After much trial-and-error, I found that the following steps worked best for me on a Windows machine once my github account had been set up:

  1. Run through Chapters 1 - 14 of happygitwithr.com and ensure that RStudio can connect to your github account.
  2. Implement chapter 15 of happygitwithr.com. Chapter 15 shows you how to set up a github folder for your site, clone it and open the R project for the site in the cloned directory. This makes the connection between RStudio and github seamless.
  3. I am hosting via netlify. Netlify has button, “New site from Git” that does exactly that. Once the website has been updloaded to github, you can use the “production deploys” section to deploy the site. Steps 2 and 3 ensure that the creation of a post, uploading to github, linking to netlify and then deploying the site is seamless.
